task_scheduler

Dart library for scheduling tasks

Scheduler

Runs the given callback after a specific duration

final Scheduler scheduler = Scheduler();

void printHello() => print('Hello');

scheduler.scheduleByDuration(Duration(seconds: 2), printHello);

Scheduler Immediate

Runs the given callback asynchronously as soon as possible

final Scheduler scheduler = Scheduler();

void printHello() => print('Hello');

scheduler.scheduleByDuration(Duration(seconds: 2), printHello);

Scheduler Peridodic

Runs the given callback periodicly for a given iteration

final Scheduler scheduler = Scheduler();

void printHello() => print('Hello');

scheduler.schedulePeriodic(Duration(seconds: 2), printHello, 3);

Scheduler By Rime

Runs the given callback at a specific date time

final Scheduler scheduler = Scheduler();

void printHello() => print('Hello');

scheduler.scheduleByTime(DateTime(2021, 09, 05, 10, 10), printHello);
